## Personal Finance Advisor

A web-based personal finance advisor that allows users to:
- Plan monthly budgets
- Track expenses
- Get investment suggestions
- Calculate SIP growth
- Analyze net worth using charts

Built using HTML, CSS, JavaScript, Tailwind CSS, and Chart.js.
